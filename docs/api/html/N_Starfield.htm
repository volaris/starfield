Starfield Namespace

The base Starfield namespace all of the core classes of the Starfield are defined here.
Classes
  ClassDescription
CustomDriver
This class is used to create named instances for configurable sets of Starfield animations that can use multiple instances of a single driver type with different settings.
DriverLoader
This class provides methods to load Starfield drivers. Use this if you want to create a new application to run the Starfield.
DriverType
The DriverType attribute is used to tag a class at compile time, drivers that connect over the network will need to supply this information when they connect.
SplitStarfieldModel
StarfieldModel
This is the core Starfield class. It defines and manages all of the properties that determine what is displayed on the Starfield
Interfaces
  InterfaceDescription
IStarfieldDriver
Interface for Starfield drivers. Implement this interface to create a new animation.
Delegates
  DelegateDescription
DriverLoaderDriverFilterDelegate
Driver filter delegate. This delegate type is used to filter the drivers based on the driver type. For example, selecting only ambient drivers.
Enumerations
  EnumerationDescription
DriverTypes
Defines the general classes of drivers the Starfield supports. This is used to allow a control application to select an appropriate driver.
